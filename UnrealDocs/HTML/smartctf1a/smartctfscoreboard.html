SmartCTF1A.SmartCTFScoreBoard

Extends
ScoreBoardTeamDeathMatch

We write this Class to show all the Smart statistics collected by
SmartCTF in a visually appealing order. We also display the the
appropriate Time Stamp (DayofWeek, Day Month Year | Elapsed Time) and
server information. TODO: Find better Textures. @author The_Cowboy @version 1A @since 1A
